# Data Privacy — Trackers & Sources

A living list of the authoritative data-privacy references practitioners rely on. Subscription/paywalled and JavaScript-only sources are listed for reference but are not crawled into the KB; the freely-crawlable primary sources (gdpr-info.eu, cppa.ca.gov) are ingested via the data-privacy crawlers.

## Core trackers (authoritative cross-jurisdiction)

- **DLA Piper — Data Protection Laws of the World** — [dlapiperdataprotection.com](https://www.dlapiperdataprotection.com/) — 160+ countries on transfers, security, breach notification, enforcement, e-marketing; free, per-jurisdiction PDFs. The gold standard. *(JS app — reference, not crawled.)*
- **IAPP — Global Privacy Directory** — [iapp.org/resources/global-privacy-directory](https://iapp.org/resources/global-privacy-directory/) — data protection authorities + legislation per jurisdiction. Also the **US State Privacy Legislation Tracker** — [iapp.org/resources/article/us-state-privacy-legislation-tracker](https://iapp.org/resources/article/us-state-privacy-legislation-tracker/).
- **OneTrust DataGuidance** — [dataguidance.com](https://www.dataguidance.com/) — deep comparative analysis. *(Subscription.)*
- **ICLG — Data Protection** — [iclg.com/practice-areas/data-protection-laws-and-regulations](https://iclg.com/practice-areas/data-protection-laws-and-regulations) — practitioner Q&A per country, annual.
- **Usercentrics — Global Privacy Law Guide** — [usercentrics.com](https://usercentrics.com/) — plain-language summaries by jurisdiction.

## Regional / national primary sources

- **EU GDPR** — [gdpr-info.eu](https://gdpr-info.eu/) — article-by-article text with recitals. *(Crawled.)*
- **European Data Protection Board** — [edpb.europa.eu](https://edpb.europa.eu/) — official guidelines + enforcement decisions.
- **UK ICO** — [ico.org.uk](https://ico.org.uk/) — UK GDPR + DPA 2018 guidance.
- **California CPPA** — [cppa.ca.gov](https://cppa.ca.gov/) — CCPA/CPRA regulator. *(Crawled.)*
- **Brazil ANPD** — [gov.br/anpd](https://www.gov.br/anpd/) — LGPD authority.
- **China CAC** — [cac.gov.cn](https://www.cac.gov.cn/) — PIPL.
- **India MeitY** — [meity.gov.in](https://www.meity.gov.in/) — DPDP Act materials.
- **Canada OPC** — [priv.gc.ca](https://www.priv.gc.ca/) — PIPEDA + provincial.

## MDM-specific privacy resources

- **DAMA-DMBOK (2nd ed.)** — the canonical data-management reference; Data Governance, Reference & Master Data, and Data Security chapters bear on privacy. *(Book.)*
- **Gartner** — Magic Quadrant for MDM Solutions + related notes cover privacy implications. *(Paywalled.)*
- **DATAVERSITY** — [dataversity.net](https://www.dataversity.net/) — practitioner archive incl. MDM for GDPR compliance.
- **EWSolutions** — [ewsolutions.com](https://www.ewsolutions.com/) — customer-MDM governance + privacy (GDPR/CCPA) best practices.
- **Vendor whitepapers** (technically solid, vendor-flavored): Informatica, Reltio, Stibo Systems, Profisee, Semarchy, Veeva (life sciences).
